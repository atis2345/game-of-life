### How to build Game of Life

```
go mod download
go build
```

### How to run Game of Life

```
./game_of_life --size=5 --iterations=3
```

#### Game supports 2 arguments:

| Name          | Type |                  Desc |
|---------------|:----:|----------------------:|
| size          | int  |          X and Y size |
| iterations    | int  |  number of iterations |

### How to run tests

```
go test
```
